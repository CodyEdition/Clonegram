# 🧪Clonegram
An Instagram app layout clone built with Java. I built it as a school project. The app is purely the layout on an Apple device, since that was the focus of the class.

## 🔬Technologies
- Java

## 🚥Running the Project

1. Install [Android Studio](https://developer.android.com/studio).
2. Open this folder in Android Studio (`File` > `Open`).
3. Wait for Gradle to finish syncing.
4. Create or pick an emulator (`Device Manager`), or plug in a phone with USB debugging on.
5. Press the green **Run** button.
